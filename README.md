# Custom Ergonomic Mouse Design & Fabrication

## Overview
This project involves designing and creating a custom ergonomic mouse shell using **OnShape** and **3D printing**. The goal is to build a functional, comfortable mouse tailored to your own hand shape while integrating the provided [Bambu mouse kit components](https://bambulab.com).

## What I Did
*   **Imported** the starter design files into OnShape.
*   **Set units** to millimeters (mm).
*   **Designed** a custom mouse base shape using sketches.
*   **Extruded** the base and built the outer shell (~25mm height).
*   **Hollowed out** the interior to fit internal components.
*   **Shaped** the mouse profile from the side view for ergonomics.
*   **Created** button wedges to interact with the mouse switches.
*   **Added** cutouts and holes for the scroll wheel and clickers.
*   **Used** shell and extrude tools to refine thickness and structure.
*   **Added** screw holes and base attachment features.
*   **Filleted** edges for smoother handling and better feel.

---

## Key Design Considerations
*   Maintained at least a **2mm gap** for internal components.
*   Ensured all sketches were **closed** for proper extrusion.
*   Avoided **intersecting** or **floating** geometry.
*   Kept overhangs reasonable for **3D printing**.
*   Made sure buttons align correctly with the **PCB**.

## Challenges
*   Getting the internal spacing right without interfering with components.
*   Designing button wedges that actually press correctly.
*   Making the shape ergonomic while still printable.
*   Avoiding weird geometry issues like overlapping parts.

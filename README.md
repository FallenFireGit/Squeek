# Custom Ergonomic Mouse Design & Fabrication

## Overview
This was my take on the Hack Club Stasis Squeak Project. Despite my using OnShape for several years, I've never really used it to design organic geometry before (I mostly use Blender for this). So it was a fun new challenge to see how fast I could design something that was functional. At some point, it might be a good idea to learn how to use Onshape's built-in tools for this, but yeah, this is my project. Here is the mouse kit used in the project [Bambu mouse kit components](https://bambulab.com).

## What I Did
*   **Imported** the starter design files into OnShape.
*   **Set units** to millimeters (mm).
*   **Designed** a custom mouse base shape using sketches.
*   **Extruded** the base and built the outer shell (~25mm height). I also used extrude (up to next) so that it could follow the natural curvature under the mouse.
*   **Hollowed out** the interior to fit internal components using Shell.
*   **Shaped** the mouse profile from the side view for ergonomics.
*   **Added** cutouts and holes for the scroll wheel and clickers.
*   **Added** screw holes and base attachment features to attach to the Bambu.
*   **Filleted** edges for smoother handling and better feel.

## Challenges
*   Getting the internal spacing right without interfering with components.
*   Designing button wedges that actually press correctly.
*   Making the shape ergonomic while still printable.
*   Avoiding weird geometry issues like overlapping parts.
